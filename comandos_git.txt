git  init . 		-> Cria um repo git na pasta atual
git status 		-> Mostra o status do trabalho
git commit -m "msg" 	-> Faz commit do que estava alterado
git add nome_arquivo 	-> Adiciona o arquivo para ser commitado
git log 		-> Mostra todo o histórico de commit


## FLUXO DE TRABALHO

1. criar o diretório do projeto
2. iniciar o git dentro do diretório (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3
